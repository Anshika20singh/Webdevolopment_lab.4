# JavaScript Console Quiz Game

This is a simple JavaScript quiz game that runs entirely in the *browser console*.  
It uses prompt() to take user input and alert() to provide immediate feedback.

## Features
- Predefined set of questions stored in an array
- Uses loops, conditionals, and functions
- Processes input using toLowerCase() and trim()
- Tracks score and displays final result
- No HTML or DOM required
